## 1. Overview
The AI Quiz App is a web/mobile-based application designed to help users learn effectively through adaptive quizzes. The system uses AI to generate questions, evaluate responses, and personalize difficulty based on user performance.

## 2. Objectives
- Provide interactive quizzes for multiple subjects
- Adapt quiz difficulty based on user performance
- Give instant feedback and explanations
- Track user progress over time

## 3. Functional Requirements

### 3.1 User Management
- User registration and login
- User profile creation
- Secure authentication

### 3.2 Quiz Management
- Select subject/topic and difficulty level
- AI-generated quiz questions
- Timed and practice quiz modes
- Multiple-choice questions (MCQs)

### 3.3 Evaluation & Feedback
- Instant answer evaluation
- Display correct answers with explanations
- Score calculation

### 3.4 Personalization
- Adaptive difficulty based on past performance
- Recommendations for weak topics

### 3.5 Progress Tracking
- Store quiz history
- Display scores, accuracy, and improvement charts

## 4. Non-Functional Requirements
- User-friendly interface
- Fast response time
- Scalable using cloud infrastructure
- Secure data storage
- High availability

## 5. Constraints
- Use of AWS free tier
- Open-source tools and libraries
- Limited budget (student prototype)

## 6. Assumptions
- Users have internet access
- Initial version focuses on MCQ-based quizzes
